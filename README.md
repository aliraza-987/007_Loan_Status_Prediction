# Loan Status Prediction 💰

Predicts whether a loan application will be **Approved** or **Rejected** using Machine Learning.

## Dataset
- 3,192 samples
- 12 features (Gender, Income, Age, Credit History, etc.)

## Model Results

| Model | Accuracy | Precision | Recall | F1 Score |
|-------|----------|-----------|--------|----------|
| **Random Forest** | **98.44%** | **100%** | **96.97%** | **0.9846** |
| Logistic Regression | 98.12% | 99.38% | 96.97% | 0.9816 |
| SVM | 49.14% | 52.38% | 16.67% | 0.2529 |

## Best Model: Random Forest - 98.44% Accuracy ✅
